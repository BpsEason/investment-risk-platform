請生成 Django model 定義檔，名稱為 PortfolioRisk。
欄位有：
- portfolio_id (UUIDField, primary_key=True, default=uuid.uuid4, editable=False)
- metric (CharField, max_length=100)
- value (FloatField)
- calculated_at (DateTimeField, auto_now_add=True)

請確保包含必要的 import 語句（如 uuid, models, timezone），為 model 定義 __str__ 方法，並設定 Meta 類以提供 verbose_name 和 unique_together。
